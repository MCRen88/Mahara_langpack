Синхронізовані групи, що зберігаються як об'єкти LDAP

Увімкніть цей параметр, якщо ваші групи зберігаються як окремі записи у LDAP.

Приклад: 

dn: cn = languagestudents, ou = групи, dc = mahara, dc = org objectClass: groupOfUniqueNames
cn: languagestudents
uniqueMember: uid = user1, dc = mahara, dc = org
uniqueMember: uid = user2, dc = mahara, dc = org
uniqueMember: cn = frenchclass, ou = групи, dc = mahara, dc = org
