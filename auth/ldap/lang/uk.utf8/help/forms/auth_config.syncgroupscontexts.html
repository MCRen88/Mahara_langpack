Синхронізація груп лише в цьому контексті

Список контекстів, де розташовані групи. Розділіть різні контексти через ';'. (наприклад, "ou = групи, o = org; ou = other, o = org ').

Якщо це поле залишається порожнім, група синхронізації cron буде повертатися до того ж списку контекстів, що й "Контексти" для розташування користувачів.
