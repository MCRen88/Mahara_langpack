Зовнішній URL галереї

Щоб вставити зовнішню галерею, потрібно скопіювати посилання до зовнішньої галереї та вставити її в текстове поле.

Можна вставляти такі зовнішні галереї:

  • Flickr, напр. https://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa, напр. http://picasaweb.google.com/user.name/AlbumName#
  • Photobucket, напр. http://s1234.photobucket.com/albums/ee123/username/ або http://s1234.photobucket.com/albums/ee123/username/AlbumName/
  • Фотогалерея Windows Live, наприклад http://cid-123456ab7890cd12.photos.live.com/self.aspx/AlbumName/photo.jpg

Примітка.

  • Щоб скористатися Flickr, адміністратор сайту повинен надати дійсний ключ API.
  • Щоб скористатися функцією Photobucket, адміністратор сайту повинен надати дійсний ключ API і секретний ключ API.
  • Щоб фотоколекція Windows Live працювала належним чином, посилання на одну з фотографій в альбомі має бути скопійовано та вставлено в текстове поле, щоб показати весь альбом.
