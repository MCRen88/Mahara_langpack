Поля для пошуку

Поля профілю, позначені як "доступні для пошуку", будуть доступні для всіх, хто увійшов в систему для пошуку.

Ім'я, прізвище та відображувана назва зазвичай завжди доступні для пошуку. Тим не менш, точна поведінка пошуку імен контролюється налаштуваннями на сторінці параметрів сайту, для "Користувачі можуть приховувати реальні імена" і " Пошук імен користувачів", а також окремі налаштування.

Інші пошукові поля профілю, описані тут, такі як адреса електронної пошти, можна використовувати для пошуку користувачів у пошуковому запиті, але вони не відображаються в результатах пошуку.
