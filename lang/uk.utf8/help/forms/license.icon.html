Піктограма ліцензії

Це може бути URL-адреса http: // ... або https: // ... зображення, розміщеного на вашому сайті або зовні.

Це також може бути ліцензія: ... URL, щоб вказати піктограми ліцензій на тему Mahara. Наприклад, URL-адреса 

 license: gfdl.png
посилається на файл 
 images / license / gfdl.png
у темі.
